# COMP3011 CW2 — Search Engine Tool

## Overview

A command-line search engine that crawls **quotes.toscrape.com**, builds an
inverted index, and supports ranked full-text search.

- **Deadline:** 8 May 2026
- **Target site:** https://quotes.toscrape.com
- **Politeness window:** ≥ 6 seconds between requests

---

## Installation

```bash
pip install -r requirements.txt
```

---

## Usage

### Build the index (crawl + index + save)

```bash
python main.py build
# optionally specify a different start URL or output path:
python main.py build --start-url https://quotes.toscrape.com --output data/index.json
```

### Load a saved index

```bash
python main.py load --index data/index.json
```

### Print all indexed documents

```bash
python main.py print --index data/index.json
```

### Search the index

```bash
python main.py find "life love" --top-n 10 --index data/index.json
```

### Verbose / debug output

```bash
python main.py --verbose build
```

## System Architecture

The system is a three-stage pipeline:

1. **Crawler** (`search_engine/crawler.py`)  
   Uses `requests` to fetch HTML and `BeautifulSoup` to parse content and
   extract internal links. Enforces a 6-second inter-request delay. Scoped
   strictly to the `quotes.toscrape.com` domain.

2. **Indexer** (`search_engine/indexer.py`)  
   Builds an inverted index mapping tokens → document IDs. Applies a
   pre-processing pipeline: lowercase → tokenise → remove stop words → stem
   (NLTK PorterStemmer). Serialises the index to JSON for persistence.

3. **Searcher** (`search_engine/searcher.py`)  
   Loads the index and scores documents against a query using TF-IDF. Returns
   top-N ranked results with URL, title, score, and a snippet.
